# Overview
Web application that connects freelancers with recruiters (project owners).


## Common
- Setup profile
- Send/Receive messages
- Receive notifications

## Recruiter
- Create/Update/Delete/Read projects
- Create/Update/Delete/Read tasks linked to a project
- Assign a task to a freelancer
- Browse freelancers
- Send invitations to freelancers to join their projects

## Freelancer
- Create/Read/Update/Delete proposals
- Read projects
- Report projects
- Update task status
- Receive invitations
- Accept/Decline invitations
